# Matrix Generator

A small browser-based Matrix-style random text generator. It creates vertical columns of randomized characters with adjustable character sets and column count.

## Run

Open `index.html` directly in a browser.

No build step, package install, or local server is required.

## Features

- Generate vertical Matrix-style text columns
- Toggle letters, numbers, and special characters
- Adjust the number of generated columns
- Horizontally scroll when output is wider than the page
- Copy the generated output as plain text
- Uses `img/logo.png` as the page icon

## Files

- `index.html` - page markup and generator logic
- `styles.css` - layout and visual styling
- `colors.js` - color helper utilities
- `img/` - logo assets
